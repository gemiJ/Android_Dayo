# Dayo
Diary sharing app<br>
This app was made in 2020.<br>

:book: Introduction
-----------
다이어리 꾸미기 정보 & 다이어리를 통한 일상을 공유하기 위한 앱.<br>
일정, 공부, 여행기록 등 사용자가 원하는 주제로 폴더를 생성하여 내용 업로드 가능.<br>
